\subsection{Questions-réponses}

Les questions les plus courantes sont répertoriées ici pour te faire gagner du temps !


\begin{description}

\item[J'ai un deuxième ordinateur, qu'est-ce que je fais ?] je vais demander une deuxième IP sur la page \url{http://frankiz/profil/demande\_ip.php} et je la lui donne.

\item[J'ai une question sur l'informatique] je post sur \ngname{br.informatique.[truc qui va bien]}

\item[J'ai perdu mon mot de passe qRezix] je vais le re-définir dans mes  \menu{Préférences} sur \fkz : \url{http://frankiz/profil/reseau.php}

\item[Je vois mon nom et pas mon pseudo quand j'ai voté à la QDJ] je mets mon pseudo sur ma fiche trombi sur la page \url{http://frankiz/profil/profil.php} (lien \menu{Préférences})

\item[J'ai plus de réseau] je vais voir la 4\textsuperscript{ème} de couverture

\item[Mon ordinateur n'a pas de nom sur le réseau] j'installe qRezix et je le configure.

\item[Je m'ennuie] je lance la télé (cf page \pageref{tele})

\end{description}
